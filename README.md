# Rautsan - Landing Page + Toko Online

Landing page bergaya Instagram dengan toko online (Rautsan Snacks + Affiliate) dan admin panel terpisah.

## Menjalankan

**Cara paling gampang:** Double-click `run.cmd`. CMD akan terbuka, dependency otomatis di-install (jika belum), lalu server jalan. Biarkan jendela CMD terbuka untuk melihat log.

- **Landing:** http://localhost:3000  
- **Admin:** http://localhost:3000/admin  

Login admin default: **username:** `admin` | **password:** `admin123!`

## Fitur

- **Landing:** Profile (foto, cover, nama, bio, tombol Donate), link cluster (icon sosmed), toko (tab Snacks / Affiliate), grid 3x3, detail produk ala IG, popup marketplace, light/dark mode.
- **Admin:** Login lokal, dashboard telemetry, edit profile, link cluster (default YouTube/TikTok/Instagram + tambah), produk Snacks & Affiliate (CRUD, foto max 4 + cover, marketplace default Shopee/Tokopedia/TikTok Shop).

Semua data disimpan di folder `data/` dan `uploads/` (lokal).
